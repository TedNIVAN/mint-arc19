# mint-arc19

## Get Started

### Install dependencies
```console
$ npm install
```

### Setup environment
```console
$ touch .env
```

Your `.env` file should look like this:
```
PINATA_API_KEY="apikey"
PINATA_SECRET_KEY="secretkey"
ALGOD_TOKEN=""
ALGOD_SERVER="https://node.testnet.algoexplorerapi.io"
ALGOD_PORT=""
MNEMONIC="secret mnemonic phrase"
```

### Run it
```console 
npm run start
```
